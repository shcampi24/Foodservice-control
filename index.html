🔁 Tareas automáticas (recurrentes)

Por defecto todo se carga de forma manual. Estas son las únicas tareas que el sistema genera solo, todos los días, porque fueron marcadas con "🔁 Repetir automáticamente" al crearlas (ej. lavado de pisos). Podés desactivarlas acá.

🔧 Integración con Mantenimiento

Estos datos se usan para pre-generar el texto del ticket y los accesos directos cuando el Admin gestiona una solicitud.
💡 Los cambios se guardan automáticamente al escribir.

📋 Plantillas de ronda

Cada plantilla queda enganchada a un sector (los mismos que ya tenés cargados arriba). Cuando alguien de ese sector entra a "Rondas", el sistema le muestra automáticamente la que le corresponde.

🔁 Rondas fijas asignadas

Estas son las rondas que se le asignaron a una persona "para todos los días" desde el botón 🔁 al asignar una ronda. Le van a seguir apareciendo automáticamente hasta que la desactives acá.

💾 Respaldo de datos

Los datos viven en este navegador. Un backup se descarga automáticamente al cerrar sesión. También podés generarlo manualmente en cualquier momento.
💡 Guardá los archivos de backup en una carpeta de red o Google Drive con el nombre de fecha que viene incluido.
